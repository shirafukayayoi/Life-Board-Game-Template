## 概要

現行の「Campus Life Game」に対して、以下の5点の改善を行う。本ドキュメントはその設計仕様をまとめたものである。

1. **サイコロ廃止 → 固定進行制への変更**
2. **ラウンド数・イベント数の拡張（16 → 48）**
3. **単位取得システムの調整（難易度緩和・マイナス削除）**
4. **イベント内容の大幅追加・刷新**
5. **プレイヤー間インタラクションイベントの追加**（※方針ペンディング中）

---

## 1. サイコロ廃止・固定進行制

### 変更内容

| 項目 | 変更前 | 変更後 |
|---|---|---|
| 進行方式 | d3（1〜3）ダイスロール | 固定：毎ラウンド1マス（1イベント）進む |
| 1ラウンドの定義 | サイコロを振って1〜3マス進む | 1ヶ月分の出来事を1イベントで処理する |

### 理由

大学生活は「何年生の何月に何が起きるか」がある程度固定されている（履修登録、試験期間、就活解禁など）。ランダムな進行よりも、時系列に沿った進行の方がリアリティと没入感が高い。全員が同じタイミングで同じ種類のイベントを迎えることで、プレイヤー間の比較・会話が生まれやすくなる。

### 実装上の変更点

- `server/index.js` の `player_roll` 処理を削除（または無効化）
- ラウンド進行を「全プレイヤーが選択完了 → 次のラウンドへ自動遷移」に変更
- `src/pages/controllerPlay.tsx` のダイスロールUIを削除し、イベント表示に直結させる
- `server/board.js` の `diceToSquares()` 関数は不要になる

## 2. ラウンド数・イベント構造の変更

### 基本設計

- **総ラウンド数：48**（大学1年4月〜4年3月の48ヶ月）
- **1ラウンド = 1ヶ月 = 1イベント**
- ブランチ構造は廃止（月ごとのイベントは全員共通）ただしイベント内の選択肢の条件分岐（`conditionalVariants`）は維持する

### 変更理由

現行の32マス構成はd3進行を前提にしているため、固定進行に変えると体験できるイベントが極端に少なくなる。月単位にすることで「いまは何年生の何月か」がプレイヤーに直感的に伝わり、ゲームへの共感度が上がる。

## 3. 単位取得システムの変更

### 基本方針

| 項目 | 変更前 | 変更後 |
|---|---|---|
| 卒業必要単位 | 110単位 | **124単位**（文部科学省基準に準拠） |
| チェックポイントの役割 | 進級判定（基準未満で留年） | **教務課からの警告のみ**（機械的ペナルティなし） |
| 留年判定タイミング | 各年度末（4回） | **卒業時（Month 47）の1回のみ** |
| 単位の減算 | あり（スキップ・休学で-2〜-8） | **廃止**（単位は増えるか変わらないのみ） |

### 単位マイナスの削除対象

以下のイベント・選択肢から `credits` のマイナス効果を削除または0に変更する：

| イベント | 選択肢 | 現状 | 変更後 |
|---|---|---|---|
| `main.json` イベント#12 | 授業をサボる（D） | `credits: -2` | `credits: 0` |
| `main.json` イベント#22 | 休学して自分を見つめ直す（A） | `credits: -4` | `credits: 0`（休学中は増えない） |
| `main.json` イベント#22 | 休学してワーホリに行く（B） | `credits: -4` | `credits: 0`（同上） |
| `threshold.json` 留年危機 | （しきい値イベント全体） | `credits: -8` | 削除または`credits: 0` |

### 124単位の配分設計

48ラウンドで124単位を達成できるよう、以下の分配ガイドラインを設ける。

**前提：**
- 単位付与イベントは全48ラウンドのうち約**25〜28イベント**に設置する
- 各イベントで「最も勉学に集中する選択肢」を選んだ場合の最大獲得量を積み上げると**約150〜160単位**になるよう設定する
- 現実的な「勉強もするが他のこともする」プレイで**124単位前後**に達するよう各イベントの数値を調整する

**単位付与イベントの目安：**

| イベント種別 | 単位付与量の目安 | 該当数（目安） |
|---|---|---|
| 前期・後期試験（★） | 最大8〜10単位、最小1単位 | 8回（年2回×4年） |
| 卒業論文提出（★★） | 最大12単位 | 1回 |
| ゼミ・研究室 | 4〜6単位 | 3〜4回 |
| 履修登録・通常授業 | 3〜5単位 | 6〜8回 |
| 資格・任意学習 | 2〜4単位 | 4〜5回 |
| その他（小） | 1〜3単位 | 4〜6回 |

**年度別の単位獲得目安（勉強中心のプレイ）：**

| 学年 | 目標取得単位 | 累計 |
|---|---|---|
| 1年 | 約25〜30単位 | 25〜30 |
| 2年 | 約30〜35単位 | 55〜65 |
| 3年 | 約30〜35単位 | 85〜100 |
| 4年 | 約24〜30単位（論文含む） | 109〜130 |

**チェックポイントの警告ライン（ペナルティなし・参考値）：**

| タイミング | 警告ライン | 表示メッセージ例 |
|---|---|---|
| 1年終了（Month 12後） | 25単位未満 | 「このままだと単位が足りなくなるかもしれません。来年は早めに取り組みましょう。」 |
| 2年終了（Month 24後） | 55単位未満 | 「少しペースが遅れています。前期から積極的に授業を取りましょう。」 |
| 3年終了（Month 36後） | 90単位未満 | 「卒業まで34単位以上必要です。4年生は最終スパートをかけてください。」 |
| 卒業判定（Month 47） | 124単位未満 | **留年エンド確定** |
